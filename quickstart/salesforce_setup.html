Set up a developer envionrment in Salesforce

Create a developer account

  1. Open the Admin console via the top menu after launching Snoweaver.

  2. Open the Projects page via the side bar.

  1. Create a new project called DEV_LEARN by updating the Name field and click Save.

  2. A confirmation message is displayed if the new project is created successfully.

  1. Press F5 or Ctrl+R to reload your browser, and the new project shall appear on the top menu.

  2. Open the new project to verify.

  1. Switch back to the Admin console for the next task
